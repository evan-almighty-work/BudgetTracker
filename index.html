Backup and Restore

Export saves all current app data to a JSON file. Import replaces all current app data with the contents of the selected JSON backup file.

Import fully replaces current data after confirmation. Use export first if you want a safety backup.

All data is saved in your browser with local storage. If you clear your browser storage or switch devices, the data will not carry over automatically unless you export and re-import a backup.
